# Running locally

If you don't know [github pages](https://pages.github.com/) yet, have a look first.

Then, follow [instructions](https://help.github.com/articles/setting-up-your-github-pages-site-locally-with-jekyll/) to install and run Jekyll locally (skip step 3).

Start jekyll:
> bundle exec jekyll serve

Then open [http://localhost:4000](http://localhost:4000) with a browser.
